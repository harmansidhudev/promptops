Privacy Policy

Last updated: March 8, 2026

PromptOps is an open-source Claude Code plugin created by Harman Sidhu. This policy explains how the plugin handles your data.

Data Collection

PromptOps does not collect, transmit, or store any user data externally.

All data generated by the plugin — evaluations, improved prompts, regression reports, comparisons, and run outputs — is stored locally on your machine in a .promptops/ directory within your project folder.

What the Plugin Accesses

When you use PromptOps commands, the plugin processes:

  • The prompt text you provide for evaluation or improvement
  • Files in your project directory that you explicitly point it to
  • Conversation history within your current Claude Code session (for the /regression and /run commands)

This processing happens entirely within your local Claude Code environment. No data is sent to external servers, APIs, or third parties by the plugin itself.

Third-Party Services

PromptOps runs inside Claude Code, which is operated by Anthropic. Your interactions with Claude Code are subject to Anthropic's Privacy Policy. PromptOps does not add any additional third-party services or data collection beyond what Claude Code itself provides.

Saved Files

The plugin saves output files to your local filesystem:

  • .promptops/evals/ — evaluation reports
  • .promptops/improved/ — improved prompt versions
  • .promptops/regressions/ — regression comparison reports
  • .promptops/comparisons/ — model benchmark results
  • .promptops/runs/ — executed outputs
  • .promptops/datasets/ — golden dataset entries (if you opt in)

These files are under your control. You can delete them at any time. If you commit them to version control, that is your choice — the plugin does not do this automatically.

Analytics & Tracking

None. No analytics, no telemetry, no usage tracking, no cookies. The plugin has no network component.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL. Since PromptOps doesn't collect data, meaningful changes are unlikely.
